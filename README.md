# Mems表头C/V测试数据分析脚本

## Install requirements

```bash
pip install -r requirements.txt
```

## Run CV

Note: Your data directory must contains two subdirectories: `C-V` and `Sec Para`.

```bash
python main.py --data /path/to/your/data
```

## OLTF example

```bash
python oltf.py
```

## Database

```bash
python data_base_manager.py
```
